Basic Structure: The <html> Tag
  • The <!DOCTYPE html> declaration helps with browser compatibility.
  • <html> tag represents the root element of an HTML page.
  • The "lang" attribute specifies the language of the web page.

Headings: The <head> Tag

Headers are created using the heading tags from h1 to h6. They help organize your content by grouping related sections together.

<head>
  &# This is a comment.
<title>My Website Title</title>
</head>

  1. Headers are created using six levels of headings from h1 to h6.
  2. Inside the <head> section, we have a title that sets the title of our webpage.
